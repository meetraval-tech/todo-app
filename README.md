# Todo App

A full-stack Todo Application built using React and Laravel.

## Features

- Add Tasks
- Update Tasks
- Delete Tasks
- Responsive UI
- REST API Integration

## Tech Stack

### Frontend
- React
- Axios
- CSS

### Backend
- Laravel
- MySQL

## Project Structure

```bash
todo-app/
│
├── todo-frontend/
├── todo-api/
```

## Installation

### Clone Repository

```bash
git clone https://github.com/meetraval-tech/todo-app.git
```

---

## Frontend Setup

```bash
cd todo-frontend
npm install
npm run dev
```

---

## Backend Setup

```bash
cd todo-api
composer install
php artisan serve
```
