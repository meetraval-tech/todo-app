✔ Controller

Handles API logic

Example:

GET todos
POST todo
UPDATE todo
DELETE todo
🔗 API Routes (IMPORTANT)
GET    /api/todos        → get all todos
POST   /api/todos        → create todo
GET    /api/todos/{id}   → get single todo
PUT    /api/todos/{id}   → update todo
DELETE /api/todos/{id}   → delete todo
